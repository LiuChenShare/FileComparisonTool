# FileComparisonTool
文件比较工具

# 相关技术及环境
VS2019 + WPF + .NET Framework 4

# 使用说明（v1.0.0.0）
本工具用于对比两个文件夹下的文件差异，以【原始文件夹】为基础，筛查【最新文件夹】中新增及修改的所有文件并提取至【目标文件夹】中

1.分别选择三个文件夹路径  
2.点击开始提取  
3.等待进度条完成后即可在【目标文件夹】中获取提取到的文件
